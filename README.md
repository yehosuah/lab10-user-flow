# MercadoCerca - User Flow Challenge

Entrega para el reto de UX Tools: crear un user flow para una app movil de supermercado a domicilio.

## Que incluye

- Flujo principal: seleccionar tienda cercana, agregar productos, revisar carrito, checkout, pago, horario y confirmacion.
- Extra credit 1: agregar nuevo metodo de pago.
- Extra credit 2: reordenar una compra anterior.
- UI interactiva en espanol: cada nodo abre su detalle UX.

## Como verlo

Abrir `index.html` en el navegador o levantar un servidor local:

```bash
python3 -m http.server 4173
```

Luego visitar `http://localhost:4173`.
