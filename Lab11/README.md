# Lab11 - Sketching, Storyboarding y Prototipo

Entrega en formato de sitio web interactivo para una app de transporte accesible
llamada **Ruta Clara**.

## Contenido

- `index.html`: presentacion completa del laboratorio.
- `styles.css`: estilos responsivos.
- `script.js`: prototipo clicable.
- `assets/bocetos.png`: tres bocetos rapidos.
- `assets/storyboard.png`: storyboard de 8 vinetas.

## Como verlo

Abrir `index.html` en el navegador o levantar un servidor local:

```bash
python3 -m http.server 4174
```

Luego entrar a `http://localhost:4174/Lab11/`.

## Tipo de presentacion elegido

Sitio web interactivo con secciones para sketching, storyboarding, prototipo y
reflexion final.
